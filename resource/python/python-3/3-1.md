$$ 第\;3\;关：多分支结构 $$

$1.\;$有如下代码，当输入$\;4\;$时，输出结果是 `3<x<=5`。

```python
x=int(input())
if x>5:
    print("x>5")
elif x>3:
    print("3<x<=5")
else:
    print("x<=3")
```

$2.\;$以下程序的输出结果是 `a=20, b=1`。

```python
a = 30
b = 1
if a >=10:
    a = 20
elif a>=20:
    a = 30
elif a>=30:
    b = a
else:
    b = 0
print('a={}, b={}'.format(a,b))
```
